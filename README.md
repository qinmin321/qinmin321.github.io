# Min's PICO-8 Arcade 🎮

Welcome to my personal gaming arcade! Here, I showcase retro-style games built for the **PICO-8 fantasy console**.

---

## 🐍 Featured Game: Snake

An action-packed, modern take on the classic retro Snake game. 

### 🕹️ [Play Snake Now!](snake/)

### Key Features:
* **Juicy Gameplay:** Featuring satisfying screen shake and custom particle explosions.
* **Golden Apples:** Rare golden treats that spawn with a 10% chance, awarding bonus points and screen-shaking excitement.
* **Persistent High Scores:** Compete against yourself! Your highest score is saved directly to your browser's local storage.
* **Dynamic Difficulty:** The game speeds up as your score increases.

### Controls:
* **Arrow Keys (⬅️ ➡️ ⬆️ ⬇️):** Move and change direction.
* **X Key (❎):** Start or Restart game (Title & Game Over screens).
* **Z Key (🅾️):** Pause / Resume gameplay.
